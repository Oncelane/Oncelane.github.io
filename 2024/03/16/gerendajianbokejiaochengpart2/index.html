终端初识

文件夹命令

进入文件夹

cd D:\typora\file

返回上一级

cd ..

跳转到根目录

cd \

查看当前路径下所有文件

dir
ls

创建文件夹

mkdir myblog

新建非空文件

echo 文件中的内容>new.txt

查看文件中的文本内容

type 文件名

删除文件(如果是del 文件夹A是删除文件夹A内的所有带后缀的文件，若文件夹A中有文件夹B，文件夹B不会被修改)

del 文件名

删除所有指定后缀的文件

del *.txt
del *.docx

删除名为file的空文件夹

rd file

删除file文件夹下的所有文件

rd file /s

生成目录树

tree

copy 路径\文件名 路径\文件名

移动文件

move 路径\文件名 路径\文件名

四、网络相关

查看ip地址

ipconfig

ping www.csdn.net

配置git

git config --global user.name "用户名"
git config --global user.email "邮箱"
git config --list

生成ssh key

ssh解释

检查是否存在SSH Key

cd ~/.ssh
ls

如果没有，就生成ssh key

ssh-keygen -t rsa -C "you_name@your_email.com"

把SSH密钥内容copy到github上
然后测试有没有成功

ssh -T git@github.com

SublimeText安装插件

  1. 安装中文
    Install package

chinese
  2. 安装Markdown的预览功能，首先需要安装相关插件：

Ctrl+Shift+p，输入Install Package回车
等待下一个输入框，输入MarkdomnPreview回车开始安装
弹出安装完成的文档，重启SublimeText

  1. SublimeText如何快速预览Markdown

Ctrl+Shift+p，输入Markdown Preview: Preview in Browser回车
出现的选项中选择Markdown，即可跳转至默认浏览器以md格式查看

  1. 自定义预览快捷键

在Preferences -> Key Bindings打开的文件右侧栏的中括号中添加如下代码，其中alt+q可以设置为任意按键；

{ "keys": ["ctrl+q"], "command": "markdown_preview", "args": {"target": "browser", "parser":"markdown"} }

Hexo安装

  1. 所有必备的应用程序安装完成后，可使用 npm 安装 Hexo。
npm install hexo

npx hexo <command>

  1. 建站

    hexo init <folder>

cd <folder>

npm install

  2. 在本地测试：

    
hexo clean

hexo g

hexo s
